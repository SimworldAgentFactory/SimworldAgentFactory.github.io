Getting Started

Docs

Get the framework running on your machine in a few minutes. Install the base package, enable the extras you need, and run a working agent loop against MiniGrid or DeliveryBench.

Overview

AgentFactory lets you combine environment adapters, perception, reasoning, memory, and reflection modules into a single agent runtime. The setup flow below keeps the defaults simple while still making it easy to enable LLM-based methods and environment-specific dependencies.

New here? Start with the installation and quick start commands below, then use the playground once the core dependencies are installed.

Installation

Clone the repository and install it in editable mode. Add optional extras depending on the environment stack you plan to use.

terminal 
# Clone
git clone https://github.com/chenjix/agentfractory.git
cd agentfractory

# Basic install
pip install -e .

# Optional extras
pip install -e ".[thor]"      # AI2-THOR support
pip install -e ".[gym]"       # Gym / MiniGrid support

Quick Start

Set OPENAI_API_KEY if you plan to use LLM-backed reasoning. MiniGrid and AI2-THOR can then be installed independently.

terminal 
# API key for llm, react, cot, tot, guided_search, or self_refine
export OPENAI_API_KEY="sk-..."

# MiniGrid
pip install minigrid

# AI2-THOR
pip install ai2thor

After installation, environments expose the usual loop you would expect: reset(), step(), and an action schema that the agent can consume.

Environment Overview

MiniGrid is the fastest way to validate an agent configuration locally. DeliveryBench is useful when you want a richer benchmark flow, while THOR is best when you need a more grounded 3D embodied environment.

MiniGrid Environments

The following presets are exposed directly in the playground and example scripts.

Name Description
MiniGrid-Empty-5x5-v0 Simple navigation
MiniGrid-DoorKey-5x5-v0 Find key, open door, and reach goal

MiniGrid-DoorKey-8x8-v0 Even larger variant
MiniGrid-MultiRoom-N2-S4-v0 Multi-room navigation
MiniGrid-MemoryS17Random-v0 Memory-focused task

Run Your First Agent

Start with a simple MiniGrid loop, then move to a more capable reasoning module once your environment and API key are configured.

terminal 
# MiniGrid - simple reasoning (no llm)
python examples/minigrid_example.py --env MiniGrid-DoorKey-5x5-v0 --no-llm

# MiniGrid - ReAct reasoning
python examples/minigrid_react_example.py --env MiniGrid-DoorKey-5x5-v0

# MiniGrid - Guided Search
python examples/minigrid_example.py --reasoner guided_search

# DeliveryBench
python examples/deliverybench_example.py --reasoning-method react

Quick Start Code

You can also compose the framework programmatically by importing the modules directly and wiring them into an EmbodiedAgent.

quick_start.py 
import os
from modules.llm import OpenAIClient
from modules.reasoning import ReActReasoning
from agents import EmbodiedAgent

llm_client = OpenAIClient(
    api_key=os.getenv("OPENAI_API_KEY"),
    model="gpt-4o-mini",
)

reasoning = ReActReasoning(
    llm_client=llm_client,
    enable_history=True,
)

agent = EmbodiedAgent(
    perception=perception,
    reasoning=reasoning,
    memory=memory,
)
