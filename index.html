Overview

Evaluations across DeliveryBench, ALFRED, MiniGrid, and RoboTHOR show that performance depends not only on the strength of individual components but also on their interactions. Modular recomposition can outperform monolithic baselines, especially in long-horizon embodied tasks that benefit from state-aware, trajectory-level memory and structured decision making. By exposing a clear design space, AgentFactory provides a foundation for agents that are more capable, more analyzable, and more scientifically grounded.

Key Findings

Demonstrating that performance is driven more by module interaction than by individual component quality.

1

Memory Architecture

Memory utility is defined by structure (e.g., action-oriented indexing) rather than raw context volume.

Selective retrieval is critical to minimize noise and ensure high-fidelity decision-making at inference.

2

Hierarchical Abstraction

Long-horizon planning requires distilled policy-level insights (summarized experiences) to avoid the computational and cognitive overhead of processing raw trajectory fragments.

3

Multi-Granularity

Hybrid modules like MemoryBank provide the highest robustness by offering concurrent access to raw data and semantic summaries, accommodating diverse reasoning styles.

4

Abstracted Memory

High-level, distilled experiences (e.g., DynamicCheatsheet) outperform raw trajectory fragments, which often introduce noise into planning-oriented reasoning.

5

Multi-Agent Resilience

Multi-agent systems (MAD) exhibit significantly lower sensitivity to memory quality. Collective reasoning and internal error-correction preserve decision quality even when retrieved cues are coarse or partially relevant.

6

Reflective Correctness

Reflection consistently improves decision-making by benchmarking candidate actions against immediate environmental feedback, showing particular efficacy in stabilizing weak reasoning-memory pairings.

7

Domain Specificity

Module efficacy is environment-dependent. Short-horizon tasks are driven by structured reasoning, while long-horizon environments (e.g., ALFRED) are bottlenecked by the need for trajectory-level episodic memory to prevent repetitive failure loops.
