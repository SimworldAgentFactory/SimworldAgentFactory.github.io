⚙ Environment

Choose an adapter (runtime wrapper) and the specific environment variant to use. DeliveryBench uses its own fixed environment.

◉ Perception

Converts raw observations into UnifiedAgentInput. Match this to your environment adapter.

★ Reasoning

LLM-based methods require OPENAI_API_KEY to be set. Simple mode needs no API key.

♦ Memory

Stores and retrieves past observations. LLM-backed backends (ACE, A-Mem, Dynamic Cheatsheet) require OPENAI_API_KEY.

↺ Reflection

Self-Refine iteratively critiques and improves the agent's reasoning output using an LLM.

Module APIs

Core method signatures used by the framework. Helpful when implementing custom modules.
