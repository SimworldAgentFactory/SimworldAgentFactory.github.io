The memory module enables the agent to store and retrieve information beyond the immediate context, supporting accumulation of experience and reuse of task-relevant knowledge over time. It consists of both episodic memory, which captures concrete trajectories, action sequences, and outcomes in forms such as raw logs or summarized experiences, and semantic memory, which encodes abstract, reusable knowledge such as structured maps, constraints, and strategies. The framework is designed to support multiple memory paradigms within a unified interface, including retrieval-based systems, structured persistent guidance mechanisms, and agent-centric memory engineering approaches. It further accommodates modern memory architectures that emphasize long-term context management and adaptive knowledge storage, and incorporates approaches where external knowledge and semantic memory are represented as maintainable and updatable agent skills.
