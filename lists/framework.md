AGENTFACTORY is a Gym-compatible framework that formalizes agents as modular compositions of five primary components: perception, memory, reasoning, reflection, and learning. The architecture functions as a discrete loop where the Perception module first normalizes heterogeneous environment data into a standardized state representation. This abstraction enables the Memory module to retrieve relevant episodic or semantic context, which informs the Reasoning module as it generates initial action proposals.

To enhance decision quality, the Reflection module critiques and refines these proposals through self-improvement loops before execution. An optional Reinforcement Learning module utilizes environmental feedback to optimize policies over time. By decoupling these functions into reusable components with standardized interfaces, the framework enables systematic analysis of component interactions and design trade-offs across diverse tasks.
